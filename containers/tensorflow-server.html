Usage Notes

This image contains a notebook server that enables access to TensorFlow through the Jupyter notebook interface. This interface can be started as follows: - Run the Docker image, publishing Jupyter's port `docker run -p 8000:8888 -it tf:2.3.1 /bin/bash` - Run the notebook server `jupyter notebook --allow-root --ip 0.0.0.0` - From a web browser, access the notebook server, using the authentication token from the notebook console. Additional Python packages are included to support TensorFlow applications. These include matplotlib, pandas, and scikit-learn.
