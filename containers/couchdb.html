couchdb

Apache CouchDB is an open-source document-oriented NoSQL database, implemented in Erlang. CouchDB uses multiple formats and protocols to store, transfer, and process its data. It uses JSON to store data, JavaScript as its query language using MapReduce, and HTTP for an API.

See couchdb.apache.org for more information

This image is built to run on the IBM Z architecture.

Usage Notes

To start CouchDB server run the below command:
docker run --name -p [port]:5984 -d icr.io/ibmz/couchdb:3.1.1

docker run --name -e COUCHDB_USER=admin -e COUCHDB_PASSWORD=password -p [port]:5984 -d icr.io/ibmz/couchdb:3.1.1

To test the CouchDB service:
curl http://[host-ip]:[port]
