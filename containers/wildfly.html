Usage Notes

Exposed Ports: 8080 9990

To boot in standalone mode: 

docker run -it icr.io/ibmz/wildfly:24.0.0.Final

To boot in domain mode: 

docker run -it --name wildfly -p 8088:8080 -p 9990:9990 icr.io/ibmz/wildfly:24.0.0.Final domain.sh -b 0.0.0.0 -bmanagement 0.0.0.0
