Usage Notes

This image includes big-endian support and is intended to be used as a base image - though using it as a node environment works fine as well.

It is worth noting that each variant of this image features a different version of NodeJS.

The focal variant includes Node version 16.15.1 - while the alpine variant includes Node version 16.16.0.
